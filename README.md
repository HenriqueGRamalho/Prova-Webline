# Prova Prática - Cadastro de Automóveis

Este é um projeto desenvolvido para o processo seletivo de estágio na Webline Sistemas. É um sistema simples (CRUD) de cadastro de automóveis desenvolvido em PHP, utilizando MySQL como banco de dados e Bootstrap para a estilização da interface.

# Tecnologias Utilizadas

* PHP 8.5.2
* MySQL
* HTML5
* Bootstrap 5

# Como Executar o Projeto

1-  Copie a Pasta do projeto (`prova_webline`) dentro do diretório `htdocs` do servidor XAMPP.

2-  Acesse a URL `http://localhost/phpmyadmin`, importe o arquivo (`banco_de_dados.sql`) e execute-o.

3-  Inicie o Apache e MySQL no painel XAMPP e acesse a URL `http://localhost/prova_webline/index.php`.
